# timelog-app
Timer app in python includes project planning and status of work updates. You can plan projects to do. You can track time spent on each project. You can track work done on those projects.<br>
<br>
First you run the project form app and save the Project, tasks and subtasks related to the project.<br>
![project form](screenshots/Project_form.png)<br>
<br>
The Timer app itself as in the screenshot below which you run to keep track of time spent:<br>
![timelog-app](screenshots/Timer.png) <br>
<br>
You decide if you want to stop the timer or let it run till completion in which case you get a notification that time is up!<br>
![notification](screenshots/Notification.png)<br>
<br>
You decide to which project/task/subtask you log this time to and record the progress made:<br>
![Timelog](screenshots/TimelogandUpdates.png)<br>
